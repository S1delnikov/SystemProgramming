Данная программа принимает два аргумента командной строки, вторым из которых должен быть текстовый файл. После этого, в одном потоке полученный файл 

шифруется, путём занесения в файл "ON.txt" символов, со сдвигом в 8 элементов; в другом потоке происходит обратный процесс - дешифровка,

записывающая данные в файл "OFF.txt". Когда два потока выполнят свои функции, в основном потоке выполняется запись в файл "results.txt" данных из

исходного файла, "ON.txt" и "OFF.txt". Далее содержимое файла "results.txt" печатается в консоль.
