Данная программа представляет собой Paint на минималках. Из функционала у неё есть: смена цветов (в том числе с использованием палитры, помимо цветов по умолчанию), 
изменение толщины кисти, очистка холста и сохранение файла в нужную директорию. Реализована при помощи Windows Forms App (.NET Framework).
